# Connect GitHub (one time)

After you create an **empty** repo on GitHub named LifePack, run these in Terminal (replace the URL with yours):

```bash
cd "/Users/ericmasmela/Documents/LifePack-site"
git remote add origin https://github.com/YOUR_USER/YOUR_REPO.git
git push -u origin main
```

If GitHub shows a different default branch name, use that instead of `main`.
